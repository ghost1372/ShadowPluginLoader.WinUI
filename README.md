## ✨简介

在`Windows App SDK`上使用插件加载器加载插件

插件内以下内容将被加载:
- [x] `C#代码`
- [x] 自定义`WinUI控件`(`Page`/`UserControl`)
- [x] 自定义`资源字典`(`ResourceDictionary`)
- [x] `图片资源`(`Assets`)
- [x] `i18n`(`Resw`)
